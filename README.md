#  Библиотечная система (Streamlit)

Веб-приложение для управления библиотекой, написанное на **Python** с использованием **Streamlit**.  
Позволяет вести учёт книг, авторов, клиентов и выдач.

## Возможности

### Для всех пользователей (логин: `user`, пароль: `user`)
- **Просмотр каталога книг** с фильтрацией по названию/автору.
- **Список авторов** с возможностью просмотра биографических данных (годы жизни, фото).
- **Просмотр книг по жанрам**.
- **Рекомендация дня** – случайная книга на главной странице.
- **Топ популярных книг** (по числу выдач).

### Для администратора (логин: `admin`, пароль: `admin`)
- **Управление складом** – добавлять/удалять книги.
- **Редактирование книг** – изменять название, автора, год, жанр, количество страниц, обложку.
- **Выдача книг** клиентам с автоматическим списанием со склада.
- **Возврат книг** с автоматическим пополнением склада.
- **Просмотр всех выданных книг** с возможностью возврата.
- **История клиентов** – просмотр всех книг, взятых конкретным читателем.
